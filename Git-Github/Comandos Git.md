# Comandos Git

[Comandos Git](https://comandosgit.github.io/)

[Comandos Git](https://gist.github.com/leocomelli/2545add34e4fec21ec16)

- Comandos iniciais: 

  - **criar novo repositório**: git init
  - **mudar de diretório**: cd 
  - **criar uma pasta/diretório:** mkdir workspace
  - **abrir o diretório e ver todas as pastas**:
    - dir - windows
    - ls - Linux
  - **ver pasta oculta**: .ssh
  - **limpar o terminal** (a tela):
    - ctrl+l - windows
    - clear - linux
  - **voltar (retroceder) uma ação**, sair da pasta: cd..
  - **autocompletar uma palavra**: tab
  - **remover o repositório:**
    - del - no windows apaga somente os arquivos da pasta
    - rmdir - no windows apaga todo o repositório 
    - rm -rf: linux para apagar todo o repositório
  - **mover o diretório**: mv
  - **sair**: q
  - para ver se tem alguma coisa **para comitar**: git status 
  - para a **primeira vez que você mandar um arquivo** para um novo repositório no github: git push origin main
  - para enviar o **segundo arquivo em diante**para o mesmo repositório: git push

  

  ***Importante***

  - cada repositório é único

  - Sempre que o repositório do github receber o primeiro arquivo, será usado o git push origin main 
  - O push é literalmente “empurrão”, o que significa que o arquivo será empurrado da sua máquina para o github
